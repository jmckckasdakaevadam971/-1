# 4. ТЗ на 3D-модели (Blender)

  - `Slot_B_NewCassette`
- `ANIM_ReplaceCycle`

## 4.2 Рекомендуемые размеры (старт)

- Кассета: 1800 × 260 × 110 мм.
- Шахта столба: 1810 × 270 × 125 мм.
- Входная воронка: расширение +25 мм на сторону.
- Сдвиг каретки между слотами: 700–900 мм.

## 4.3 Визуальные состояния (обязательные)

1. `STATE_MISSING` — кассета отсутствует.
2. `STATE_OFF` — кассета установлена, питание выключено.
3. `STATE_REPLACE` — авария, требуется замена.
4. `STATE_IN_PROGRESS` — идет операция дроном.
5. `STATE_OK` — нормальная работа после замены.

## 4.4 Экспорт

- Формат: `GLB`.
- Точка pivot кассеты: геометрический центр для корректного монтажа.
- Точки привязки для анимации:
  - `SOCKET_ENTRY`
  - `SOCKET_LOCK`
  - `DRONE_SLOT_A`
  - `DRONE_SLOT_B`

## 4.5 Сценарий анимации

- Фаза 1: подлет и стабилизация.
- Фаза 2: захват старой кассеты (Slot A).
- Фаза 3: извлечение вверх.
- Фаза 4: сдвиг каретки.
- Фаза 5: установка новой кассеты (Slot B).
- Фаза 6: подтверждение питания и уход.
